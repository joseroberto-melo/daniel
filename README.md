# Positiva BI — Sistema de Automação e Dashboard Comercial

Sistema automatizado para consolidação de dados de vendas, ligações e cotações da Positiva Administradora.

## Estrutura
- `scrapers/` — Robôs de extração (CallBox, Portal Positiva, Planilhas)
- `database/` — Integração com Supabase
- `dashboard/` — Painel de BI em Streamlit
- `.github/workflows/` — Automação diária via GitHub Actions
